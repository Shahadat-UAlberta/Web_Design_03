What is HTML?

Answer: HTML means Hyper Text Markup Language. HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content. HTML elements are the building blocks of HTML pages HTML elements are represented by tags. In another word HTML is a tag based language.

What is HTML element?

Answer: HTML element is a start tag and an end tags with content in between. There are two types of HTML elements. They are:

  1. Block element: Block element always starts with a new line. Some block elements are <p>, <h>, <ol> tag etc.
  2. Inline element. Inline elements stays side by side. Some inline elements are <b>, <strong>, <q> tag etc.
